\section{Overview}

In the first part of this assignment, you will experiment with different reward functions in MuJoCo MPC (MJPC), 
an interactive application for real-time predictive controlwith MuJoCo. You will design a reward function to achieve a desired agent behavior and observe 
how different reward functions impact agent behavior. In the second part of the assignment, you will solve a realistic robot task, 
in which the agent needs to pick up a tool and use it to hammer a nail. You will implement the main parts of an actor-critic algorithm and explore some of the design choices of the algorithm.
